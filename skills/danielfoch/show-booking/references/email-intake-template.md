# Email Intake Template

Use this format when requesting bookings by email or pasted text:

```
Client: First Last
Timezone: America/Toronto
Preferred windows: Tue-Thu 10:00-16:00, avoid school pickup 15:00-16:00
Constraints: 2-hour notice minimum, no weekends

Listings:
- 12 Example Ave, Toronto ON | MLS #C1234567 | Office phone: (416) 555-0100
- 99 Demo St, Mississauga ON | MLS #W7654321 | Office phone: 905-555-0123
```

Minimum required fields per listing:
- Address
- Office phone

Optional:
- Listing ID
- Agent/office name
- Notes and special access instructions
